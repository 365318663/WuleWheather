唔乐天气
====
本人的本科毕业设计。社区模块，用户登录模块、城市等信息采用与本地数据库交互实现。出行建议、新闻功能、天气信息为爬取的中国天气网的信息。图片目录包含了几乎所有软件界面的截图。
